About

This site was made as part of the cs50 final project. It took me a while. Before CS50 I had used html and xml at my work. And, had worked through Automate the Boring Stuff and the Jose Portilla udemy course. I had done a few little projects myself but these were mainly fairly straight forward utility scripts or automating something boring at work.

Ubuntu

I had dabbled briefly with the Odin Project to get the basics of html and, as recommended, I had a set up an Ubuntu Virtual Machine. I created this site on that machine. My main experience with Linux commands had been through using git bash on windows, and using whichever commands still work through Windows Powershell so this was sometimes tricky.

Vim

As I use Vim, I must talk about Vim. It takes far longer than I would like to start messing around with the vimrc. But, once you've got something basic you can generally leave it. I mainly used the inbuilt vimtutor to get comfortable with the keybindings, and whilst there was a learning curve, I got comfortable using this surprisingly quickly.

CSS

CSS is cascading style sheets, these are used to "style" the site. I deliberately did not use any css frameworks for this, as I wanted to actually practise vanilla css.

Flask

Flask is a python framework for controlling how a web app works. This is where the majority of the code for this was written.

Sqlite

SQL is a database query language. The database stores every book recommendation that is made and these can be searched through and filtered via the history page.
