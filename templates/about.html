About

Books Anon serves as a place to recommend a book and look for book recommendations. Books are submitted to the database by users, and the application asks for no personal data from the user. The site was made as my final project for the CS50 course. It is an Open Source Application and, if you wish, you can review the code on github.

The book data is collected from the Open Library API. As biblographical data can be a little tricky, results are filtered so you will only get results in your search if it has a value for:

  • Title - as in the book title
  • Author - as in a valid book author
  • A median number of pages for all editions associated with a work
  • A first publish date - as in the first year of publication
  • A cover image - as in a cover image of the book

If your query is vague or just returns a lot of results then, the only the first ten valid books will be returned. You then select the one that you would like to submit to the database. You can leave a review of the book but only if you want to.

Any bugs, or enhancement requests can be submitted via the github page.
